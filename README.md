# Complex Painting

Apply a given complex function to an image. Plot the image ovelayed with the contour lines of the map. Plot the resulting complex plane once the map is applied to each pixel of the image.

### To run 
./complex_pain.py

### Requisites

  - numpy
  - matplotlib
  - cplot
